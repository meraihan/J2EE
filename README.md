# Library_NSTU
NSTU Central library management	system is an Desktop Application for Noakhali Science	& Technology University	You	will get all information about all departmental	books of NSTU Such as
 - All departmental	books with many copy’s
 - Each	book contain a specific	call number
 - Same	book contain same accession	number
 - Every student can borrow	one	book at	a time
 - after returning a book then he can borrow another book
 - only	library	administrator use this for maintenance
 - it can records all students information besides	book info
